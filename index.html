About Me

My desire to study software engineering came about through my fascination on how software works on different platforms. To be precise, what I find to be more interesting is how networking devices such as routers and switches, are programmed. These interests developed while I was taking my CCNA certifications and I had the urge to understand the programming behind these devices.

This made me to join Moringa School , which is an intensive coding bootcamp. During my time at Moringa school, I specialized in Android, Java being the main programming language and on top of this, I learned how to build full-scale web applications using spark for my UI and java for my backend.
